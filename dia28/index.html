Ejericio 1

click

Se dispara cuando el usuario hace clic (ratón o equivalente táctil) en un elemento.

dblclick

Se dispara al hacer doble clic sobre un elemento.

mouseover

Cuando el cursor entra en el área de un elemento (incluye hijos).

Pasa el ratón por aquí

mouseout

Cuando el cursor sale del área de un elemento (incluye hijos).

Saca el ratón de aquí

mousedown

Se dispara cuando se pulsa un botón del ratón sobre un elemento.

mouseup

Se dispara cuando se suelta el botón del ratón tras pulsarlo.

mousemove

Se dispara continuamente cuando el cursor se mueve dentro del elemento.

Mueve el ratón aquí

keydown

Cuando se pulsa una tecla (dispara al presionar, antes del carácter final).

keyup

Cuando se suelta una tecla (útil para leer el valor final).

keypress

(Obsoleto en algunos navegadores) Se dispara al pulsar una tecla que genera un carácter.

input

Se dispara cuando cambia el valor de un input mientras el usuario escribe.

change

Se dispara cuando el valor de un campo cambia y pierde el foco (o al seleccionar opción).

submit

Se dispara cuando se envía un formulario (ideal para validar o prevenir envío).

focus

Se dispara cuando un elemento recibe el foco (p. ej. input). No burbujea.

blur

Se dispara cuando un elemento pierde el foco. Útil para validar al salir.

resize

Se dispara cuando la ventana del navegador cambia de tamaño.

Cambia el tamaño de la ventana para ver el evento en el registro.

Se dispara cuando se desplaza el contenido de un contenedor o la ventana.
